## ⚙️ Key Commands

| Command | Action |
|:---|:---|
| `.menu` | Full command list |

| `.ai [text]` | Chat with AI |

| `.epl` | Live EPL scores |
| `.tagall` | Tag all group members |
| `.antidelete on` | Recover deleted messages |
